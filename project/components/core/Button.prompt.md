Primary action control — use for the main action in a view; pair an `iconLeft` with a short verb label.

```jsx
<Button iconLeft="plus" onClick={create}>New item</Button>
<Button variant="secondary">Cancel</Button>
```

Variants: `primary` (brand fill), `secondary` (outline), `ghost` (text), `danger`. Sizes `sm | md | lg`. `fullWidth` for sheets/forms.
